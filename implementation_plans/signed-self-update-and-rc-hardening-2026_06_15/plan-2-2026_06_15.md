See `outline.md`. Milestone: **beta.1**. Prerequisites: none for code, but **D4 (air-gap omit) is locked**
(zero `fs` ⇒ no `artifacts.json`, byte-identical). Branch off main, stack plan-3/4 on it.

## Goal
Add a typed `FetchSettings` channel threaded through the single shared render path, so plans 3/4 fill fields
rather than re-opening the `render.All` signature — with a HARD byte-identical gate when `fs` is zero. No
template/behavior change in this plan.

## Implementation steps
1. Define in `render.go`:
   ```go
   type Artifact struct { Asset, SHA256 string }
   type FetchSettings struct {
     GithubProxy, MimicVersion, MimicReleaseBase string
     MimicDebs map[string]Artifact // "<codename>-<arch>"
     AgentVersion, AgentMinVersion, AgentReleaseBase string
     AgentBins map[string]Artifact // "linux-<arch>"
   }
   func All(result *compiler.CompileResult, keys map[string]compiler.KeyPair, fs FetchSettings) error
   ```
   Define ALL fields now (plans 3/4/9 fill them) so the signature never re-opens.
2. Thread zero-value `FetchSettings{}` through the 4 callers. `compile.go:147` reads
   `GetSettings`→`DefaultSettings()` fallback (existing `loadSettings` pattern) and maps it into `fs`
   (fields stay empty until plan-3/4 populate `ControllerSettings`).
3. Add the matching empty-safe fields to `InstallScriptConfig`; thread `fs` →
   `RenderInstallScriptSigned`/`RenderClientInstallScriptSigned`/`buildInstallScriptConfig`. **No template
   edits** — fields exist but are unused this plan.
4. Commit; push; PR.
