1. **Full validator table (D5):** charset/format validation for `endpoint_host` + `public_endpoints[].host`
   (injection-adjacent → root scripts), `mtu` range, `router_id` MAC-48, `extra_prefixes[]` CIDR, the four
   `ssh_*` strict charset. Each with a coded error (`apierr`/validator code) + en/zh message. **Flip the
   corresponding validation.md rows to `schema`/`semantic` HERE** (plan-1 left them `none-yet (post-beta)`).
   Validation rejects BEFORE render → does NOT move plan-2's byte-identity fixtures.
2. **`/enroll` throttle:** per-IP brute-force limiter (reuse `loginratelimit.go`).
3. **Graceful shutdown:** `signal.Notify(SIGTERM/SIGINT)` → `http.Server.Shutdown(ctx)` draining BOTH the
   operator and agent listeners (timeouts already exist — do NOT relist those).
4. **Audit-log bound:** cap entries + stop the full-file RMW per append (`readAudit`→`writeJSONAtomic`);
   ring/rotate or append-mode.
5. **Docker HEALTHCHECK** → `/api/health`.
6. **Node/edge count bound:** reject oversized topologies at compile entry (distinct from the body-size cap).
7. **Schema-version migration branch:** read-first confirm dormant-vs-absent, then make the load path branch
   on `AllocSchemaVersion` and upgrade old stored topologies.
8. **FileStore SPOF:** **note-only** in code + `docs/spec/controller/store.md`; the mutex/200ms-poll fix is
   DEFERRED to rc.2/GA.
9. Commit per sub-step; push; PR.
