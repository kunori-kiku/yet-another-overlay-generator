## Implementation steps

1. **allocation-stability.md** — under "Canonical link key", add the linkKey extension:
   `linkKey(edge) = pinKey(a,b)` for the pair's **primary-class** link (all edges with
   `role != "backup"` unify to one link), `pinKey(a,b) + "#" + edge.ID` for every backup edge —
   ALWAYS, even when the backup is the pair's only edge. (Refinement over the draft's
   lowest-edge-ID rule: edge UUIDs sort randomly, so a new backup could steal the bare pinKey
   and rename the deployed primary's interface — identity must never migrate on growth.) State
   the no-drift theorems: (a) single-edge pairs reduce to bare pinKey → fresh-compile
   order/values unchanged; (b) pinned edges are immune to re-keying because pins are reserved
   verbatim from the edge. Rewrite stage-1 "collapse reverse/duplicate edges" to the unify rule
   (Decisions #8). Role flips are deliberate identity changes (interface renames; values survive
   via pins); warn when a pair ends up with no primary-class link.
   Re-key the duplicate-pin validation rule from "two distinct links" to "two distinct linkKeys".
   Note on I9: delete/re-add idempotence holds per link identity; re-adding a backup mints a new
   edge ID → a fresh gap-fill (documented, accepted).
2. **data-model/edge.md** — add `role` field (`primary`|`backup`|empty, omitempty); document
   multi-edge pairs, per-edge `pinned_*` ownership, and that the ⚠ accidental-duplicate state is
   "roleless second edge in the same direction".
3. **artifacts/naming.md** — edge-aware naming: canonical edge keeps `WgInterfaceName(remoteName)`
   byte-identical (zero fleet rename churn); non-canonical edges use
   `"wg-" + clean[:8] + sha256(remoteName + "|" + edge.ID)[:4]` (3+8+4=15 budget). New invariant:
   per-node interface-name uniqueness across all peer interfaces, validator-enforced (collision =
   compile error, same contract class as N3).
4. **api/wire-contract.md** — Edge table row for `role` (Go `Role` / TS `role?` / UI editor:
   RightPanel toggle / round-trip yes).
5. **artifacts/babel.md** — cost resolution: explicit Priority/Weight → else `role=backup` → 384
   (4× wired default 96) → else 0 (babeld default). Equal-effective-cost warning semantics.
6. **compiler/validation.md** — new rows: role enum check; at most one `role=primary` per pair;
   equal-cost multi-link warning; D71 re-scope (roleless same-direction extra → warning suggesting
   `role=backup`); per-node interface-name uniqueness error. Client exactly-one-edge rule
   unchanged (cite `semantic.go:480-492`).
7. **security/security.md** — one paragraph: parallel tunnels share node keys (acceptable on
   distinct devices/ports, cite Pro Custodibus); per-edge keypairs recorded as future escape
   hatch, NOT implemented.
