# plan-2 — live mimic condition (re-probe the mimic@ unit each heartbeat)

**Goal:** the panel's `mimic` Node Condition reflects the LIVE unit state, not a frozen deploy-time
breadcrumb. `systemctl stop mimic@eth0` (or the unit failing/flapping at runtime) must show up.

Reads from specs: `docs/spec/artifacts/mimic.md`; the beta.10 telemetry Sampler framework.

## Changes (`internal/agent`)
- Mirror the wireguard-condition pattern (`conditions_wireguard.go`: `wgShowFn` +
  `wgShowTimeout` + `exec.CommandContext`): add a **timeout-guarded, indirected** probe
  `mimicUnitActiveFn(egress string) (active bool)` that runs `systemctl is-active mimic@<egress>` (or
  `systemctl show -p ActiveState`) under a bound timeout, so a wedged systemctl NEVER blocks the
  heartbeat. Indirected (a package var) so tests inject without touching systemd.
- `readMimicCondition` (`mimic_condition.go:80`): after reading the breadcrumb (outcome + `egress`),
  RECONCILE with the live unit — when the breadcrumb outcome expects mimic RUNNING (`active` /
  `native_downgraded_skb`) but `mimicUnitActiveFn(egress)` reports NOT active, return a live `warn`
  condition (a new reason, e.g. `mimicReasonStopped` → "Mimic unit not running (was active at deploy)")
  instead of the stale "active". Outcomes that don't expect mimic running (`fell_back_to_udp`,
  `module_unavailable`, etc.) are unchanged (the breadcrumb is authoritative there). Egress empty →
  fall back to the breadcrumb verbatim (no live check possible).
- New `mimicReasonStopped` + `mimicCond.*` i18n (en+zh) + the `REASON_LABEL` map entry
  (`frontend/src/lib/mimicCond.ts`) + the `mimicCond.conformance.test.ts` reasons list (the rc.2/3
  lesson: keep that list in sync).
- Tests: `classifyMimic`/reconcile unit tests (breadcrumb active + unit inactive → stopped; active +
  active → active; fell_back + inactive → fell_back); the FE mimicCond reason-map conformance.

## Verify + branch
Go build/vet/test (agent) + gofmt; FE build + vitest + lint (i18n parity). No install.sh/goldens
impact. Branch `feat/mimic-condition-live`.
