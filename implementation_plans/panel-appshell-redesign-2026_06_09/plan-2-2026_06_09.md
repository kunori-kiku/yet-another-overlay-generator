Turn the conditional-render scenes into deep-linkable routes under the shell, retiring `viewMode`. Sidebar/topbar become `NavLink`s; the canvas mounts only on `/design`.

## Implementation steps
1. **Routes** under `<Shell/>`: `/overview` (controller landing, thin v1 — counts + last deploy), `/design` (canvas + today's topology scene), `/fleet` (NodeRegistry + EnrollmentFlow), `/deploy` (DeployBar + compile-result preview), `/security` (TwoFactor/Passkey/keystone/AuditLog + renamed topology "Compile History"), `/settings` (Connection/Bootstrap/Mode/Appearance shells). Index route → redirect to mode's landing (`/design` local, `/overview` controller).
2. **Scope `ReactFlowProvider`** to the `/design` route element so the canvas only mounts there.
3. **Sidebar/Topbar `NavLink`s** with active styling (accent). Visibility per mode (mode still `useState` until P4 — read it where needed; full persistence in P4).
4. **Retire `viewMode`/`setViewMode`** from `topologyStore`; replace call sites with navigation. Keep selection state in the store.
5. **Rename** topology `AuditView` user-facing label to "Compile History" (avoid collision with controller Audit log); both under `/security` or `/design` per outline.
6. i18n for all nav labels (EN/ZH).
