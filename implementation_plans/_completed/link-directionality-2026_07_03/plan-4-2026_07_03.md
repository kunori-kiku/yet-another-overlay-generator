## Implementation steps

1. **CHANGELOG**: `## [2.0.0-beta.18] - <date>` — Added (per-edge `link_direction`: the field, the
   six validation codes, the EdgeEditor select + directional canvas style, the realtunnel c4
   proof), Fixed (the reverse-peer race bypassing relay/accelerator endpoints — the owner's live
   symptom), plus footer `[Unreleased]`/compare-link roll. The delta covers EVERY merged PR since
   the beta.17 tag (check `git log v2.0.0-beta.17..HEAD --oneline` — a release CHANGELOG covers the
   whole range, beta.9 lesson).
2. **Local verify** (full suite):
   `~/.local/go/bin/go build ./... && go vet ./... && go vet -tags airgap ./... && go test -race ./... && go test -tags airgap ./...`
   + `cd frontend && npm run lint && npm run build && npx vitest run`.
3. **CHANGELOG PR** → review → merge → `main` green.
4. **Tag**: `git tag -a v2.0.0-beta.18 -m "v2.0.0-beta.18"` on the merged tip → `git push origin
   v2.0.0-beta.18` → Release workflow green; verify the asset set (per-arch `yaog-agent-*` +
   `.sha256` sidecars, bundles, airgap servers, local-design zip — 29 assets at beta.17 parity) and
   the published binary's `version` stamp.
5. **Promote**: `gh release edit v2.0.0-beta.18 --latest --prerelease=false` after asset
   verification (beta.17 demotes automatically).
6. **Closeout**: STATUS.md (subject delivered; owed = owner fleet smoke → then pre-rc1-hardening
   plan-11 cuts rc.1); outline status table rows → done + PR numbers; project memory entry
   (update `MEMORY.md` pointer); subject folder STAYS in `implementation_plans/` until the smoke
   passes (archive to `_completed/` rides the rc.1 session).
7. **Owner smoke handoff** (paste into STATUS/memory): set the fleet's accelerator edge to
   `A → B` (forward), Deploy, then on B `wg show <iface>` — the peer for A must show NO configured
   endpoint until A's handshake arrives, and the roamed-in source must be the relay's egress, not
   A's direct IP; reboot/restart in BOTH orders (A-first, B-first) and confirm the path never goes
   direct.
