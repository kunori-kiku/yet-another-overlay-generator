# plan-2 ‚Äî per-node egress-interface override

**Goal:** let an operator override the auto-detected mimic egress interface per node (multi-homed /
policy-routing / `wan0` boxes), default empty ‚â° auto-detect (byte-identical to today). Answers the
owner's "the default interface better be configurable" ask.

Reads from specs: `docs/spec/artifacts/mimic.md` (egress detection), `docs/spec/data-model/*`.

## Changes
- **Model:** `internal/model/topology.go` ‚Äî `MimicEgressInterface string json:"mimic_egress_interface,omitempty"`
  beside `XDPMode` (per-node; doc comment: "" ‚â° auto-detect from the default route). TS mirror in
  `frontend/src/types/topology.ts`; add to the node-omitempty list if one is drift-pinned.
- **Renderers** (`script.go` node + client, byte-mirrored in `script.ts`): where `MIMIC_EGRESS_IF` /
  `MIMIC_EGRESS_IP` are derived (L835/L1706) AND the Phase-0 cleanup detection (L194/L1295), prefer
  the override when set: `MIMIC_EGRESS_IF={{ shq .MimicEgressInterface }}` and derive the IP from THAT
  interface (`ip -o -4 addr show dev "$IF" | awk '{print $4}' | cut -d/ -f1 | head -1`), else the
  existing `ip route show default` / `ip route get 1.1.1.1` auto-detect. Gate on a precomputed bool
  (`MimicEgressOverride`) ‚Äî NO `{{ if eq }}` (TS engine has no `eq`). Empty override ‚áí the existing
  branch renders byte-identically (assert zero golden churn).
- **Validation:** a light schema check (`internal/validator` + `validator.ts`) ‚Äî if set, the value is
  a plausible interface name (non-empty, `^[A-Za-z0-9._-]+$`, length bound); `validation_node_mimic_egress_interface_invalid`
  + `error.*` i18n (en+zh). Only meaningful when the node has a mimic (tcp) link ‚Äî otherwise inert.
- **Panel:** `NodeEditor.tsx` ‚Äî an optional text input beside the XDP-mode control (label + placeholder
  "auto-detect (default route)"), `data-testid="node-mimic-egress-interface"`, written via
  `updateNode`. i18n `nodeEditor.mimicEgressInterface*` (en+zh).
- **Conformance:** regen both corpora + drift (install.sh + model field-lists changed); a fixture with
  an override set (golden proves `MIMIC_EGRESS_IF=<override>` + interface-derived IP) and the zero-churn
  assertion for the empty case.
