# plan-1 — Backend: prefix split + custody enforcement
<!-- drafted: 2026-06-12 -->
Outline: [`outline.md`](outline.md) · Prerequisites: none (first plan)

## Goal

Replace the single shared `YAOG_CONTROLLER_PATH_PREFIX` with two independent prefixes —
`YAOG_OPERATOR_PATH_PREFIX` (operator/panel mux, `:8080`) and `YAOG_AGENT_PATH_PREFIX` (agent
mux, `:9090`) — clean break per D3. Enforce the key-custody principle at the store boundary:
`POST /update-topology` rejects payloads carrying any non-empty `wireguard_private_key` with
400 (D4). Add the missing audit entries (`update-topology`, `promote`) and a startup log line
naming both mounted base paths.

Reads from specs: controller-operator-api, controller-agent-api, controller-store

## Implementation steps

1. **Env + handler split.** In `cmd/server/main.go`: replace `envPathPrefix` with
   `envOperatorPathPrefix = "YAOG_OPERATOR_PATH_PREFIX"` and
   `envAgentPathPrefix = "YAOG_AGENT_PATH_PREFIX"`. In `handler_controller.go`: replace the
   single `pathPrefix` with `operatorPrefix`/`agentPrefix` + normalization (reuse the existing
   Trim logic), e.g.:
   ```go
   func (h *ControllerHandler) SetOperatorPathPrefix(prefix string)
   func (h *ControllerHandler) SetAgentPathPrefix(prefix string)
   func (h *ControllerHandler) operatorBasePath() string // operatorPrefix + "/api/v1/controller/"
   func (h *ControllerHandler) agentBasePath() string
   ```
   RegisterOperatorRoutes uses operatorBasePath; RegisterAgentRoutes uses agentBasePath.
   Grep for ALL old-env/`SetPathPrefix`/`basePath()` consumers before deleting
   (`grep -rn "YAOG_CONTROLLER_PATH_PREFIX\|SetPathPrefix\|basePath" --include="*.go" --include="*.yml" --include="*.md" .`)
   — if anything unexpected appears (workflows, scripts), STOP per outline insertion-point plan-1.5.
2. **Bootstrap composition** (`handler_bootstrap.go:140-143`): controller base =
   `TrimRight(PublicAgentURL,"/") + agentPrefix`.
3. **SPA guard** (`static.go`): the `/api/v1/controller/` containment check already covers any
   prefix; verify and add a comment naming both prefixes.
4. **Startup log** (in `serveController`, after EnableController):
   `log.Printf("controller: operator routes at %s (addr %s); agent routes at %s (addr %s)", ...)`
   — prefixes are secrets-by-obscurity only; logging them locally is acceptable (log goes to the
   operator's own container) — note this reasoning in a comment.
5. **Custody rejection** in `HandleUpdateTopology`: after `json.Valid`, unmarshal into
   `model.Topology` and reject if any node has non-empty `WireGuardPrivateKey`:
   `writeError(w, http.StatusBadRequest, "topology carries WireGuard private keys; controller mode is zero-knowledge — strip them client-side (key custody)")`.
   Unmarshal-then-check (not substring match) to avoid false positives in names/notes. A payload
   that fails to unmarshal into model.Topology but passed json.Valid: keep current verbatim-store
   behavior for forward-compat of unknown fields? NO — decide: unmarshal errors → 400 too
   (stricter, but update-topology bodies are always panel-produced model.Topology; document).
6. **Audit entries**: append `update-topology` (with version) and `promote` (with generation)
   audit entries following the `pin-operator-credential` pattern at `handler_controller.go:1176-1183`.
7. **Config surface**: docker-compose.yml env block (two commented lines replacing one);
   `docs/spec/controller/docker.md`, `controller-api.md` §two-ports, `persistence.md` (note the
   enforcement, full doc pass stays in plan-7); root README env table row swap (minimal —
   full rewrite in plan-7).
8. **Tests** (see below), then: commit per substep, e.g.
   `git commit -m "feat(controller): split operator/agent path prefixes (YAOG_OPERATOR_PATH_PREFIX / YAOG_AGENT_PATH_PREFIX)"`,
   `git commit -m "feat(controller): reject private-key-bearing topologies at /update-topology (custody enforcement)"`,
   push branch `plan-1-prefix-split-custody`, open PR.
9. **Rollout step (live deployment)**: PR description includes the migration snippet for
   overlay.kunorikiku.com (`.env`: rename var; `docker compose up -d`; verify curl matrix).
