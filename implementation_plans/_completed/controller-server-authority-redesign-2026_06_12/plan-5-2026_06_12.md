   `export function stripPrivateKeys(topo: Topology): { topo: Topology; stripped: number }`

   stripped copy; if `stripped > 0`, surface a one-line notice ("N private keys removed before
   upload â€” controller mode is zero-knowledge"). The server 400 (plan-1) should now be

   controllerâ†’local transition with a confirm dialog listing concretely: "WireGuard private

   `purgeModeBoundaryState()` in topologyStore â€” clears `wireguard_private_key` +

   localâ†’controller direction needs no purge (hydration overwrites at login) â€” but flipping TO

   recovery backstop; the guard is the prevention layer.
5. i18n for all new strings; branch `plan-5-mode-boundary-custody`; per-substep commits; PR.
