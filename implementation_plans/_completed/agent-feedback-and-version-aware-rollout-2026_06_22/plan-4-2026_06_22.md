# plan-4 — Per-link mimic fallback: model + compiler + validator + global default setting

> Part of [outline.md](./outline.md) — subject: agent-feedback-and-version-aware-rollout.

## Prerequisites

- **None for the model + compiler + validator + setting in THIS plan.** The pure pipeline
  changes here stand alone; Track A (plan-1/-2/-3) and Track C (plan-7/-8/-9) are orthogonal.
- **Pairs with plan-5 for the MECHANISM.** plan-4 only adds the *policy* (a resolved per-link
  string threaded into `PeerInfo`). plan-5 consumes `PeerInfo.MimicFallback` to branch the
  install script + emit the mimic condition; plan-6 adds the UI. This plan ships the policy
  end-to-end through the pure compiler/renderer-input pipeline with NO behavior change to any
  rendered artifact yet (the resolved field is carried, not yet read by any renderer template).
- This is the **Track B start** per outline §7.

## Goal

Add a per-link mimic-fallback **policy** end to end through the pure compiler pipeline, plus a
fleet-wide default. `Edge` gains a tri-state `mimic_fallback` (`""` inherit / `"udp"` / `"none"`);
`ControllerSettings` gains `MimicFallbackDefault` (the policy a link inherits); the compiler
resolves the EFFECTIVE per-link policy via a pure `resolveMimicFallback()` helper and threads the
result into `PeerInfo.MimicFallback` (and `ClientPeerInfo.MimicFallback`); the validator accepts
the field (enum-checked, only meaningful on `transport:"tcp"` edges). The field is pure policy: it
feeds the renderer-input struct, **NEVER** the allocator — allocation-stability and the Go↔TS
conformance byte-set must stay byte-exact for unchanged inputs. The shipped global default is
**OFF / `"none"` (fail-closed)** per D1; the operator can flip it (flag for owner confirm at review).

### Step 2 — Compiler: pure `resolveMimicFallback()` helper + constants

In `internal/compiler/peers.go`, beside `isMimicEdge`/`effectiveMTU` (:161-185), add closed-enum
constants and a pure resolver. This is the **policy contract** the perpetual test guards:

```go
// Mimic-fallback policy values (the closed enum for Edge.MimicFallback and the resolved
// PeerInfo.MimicFallback). "" on an edge means "inherit the fleet default"; the RESOLVED value
// on a PeerInfo is always one of mimicFallbackUDP / mimicFallbackNone (never "").
const (
	mimicFallbackInherit = ""     // edge-level only: defer to the fleet-wide default
	mimicFallbackUDP     = "udp"  // fall back to plain UDP if mimic provisioning fails
	mimicFallbackNone    = "none" // fail closed; do not fall back (the shipped default, D1)
)

// resolveMimicFallback computes the EFFECTIVE per-link mimic-fallback policy:
//   effective = edgePolicy            if edgePolicy is "udp" or "none";
//             = defaultPolicy         if edgePolicy is "" (inherit) and defaultPolicy is set;
//             = mimicFallbackNone     otherwise (the fail-closed floor — D1, shipped default OFF).
// PURE: deterministic in its two string args only, so the Go and TS ports stay byte-identical and
// it never touches allocation. An unrecognized defaultPolicy (defensive) also floors to "none".
func resolveMimicFallback(edgePolicy, defaultPolicy string) string {
	switch edgePolicy {
	case mimicFallbackUDP:
		return mimicFallbackUDP
	case mimicFallbackNone:
		return mimicFallbackNone
	}
	// edge inherits: take the fleet default, flooring anything non-"udp" to "none".
	if defaultPolicy == mimicFallbackUDP {
		return mimicFallbackUDP
	}
	return mimicFallbackNone
}
```

Rationale for flooring to `"none"`: D1 ships the default OFF (fail-closed), and the resolved
`PeerInfo.MimicFallback` is always a definite `"udp"`/`"none"` so plan-5's script branch never has
to re-interpret `""`. With `defaultPolicy==""` (the air-gap/CLI and conformance default) every link
resolves to `"none"` — which plan-5 renders identically to today's fail-closed mimic install, so
**no rendered artifact changes** in this plan.
