## Implementation steps

1. **edge.md** — rewrite the `transport` row + add a "TCP transport (mimic)" subsection:
   `tcp` means the link is wrapped by mimic (eBPF UDP→fake-TCP) for UDP-hostile networks; keyless
   (WG provides crypto), so **no new edge field**; per-interface, MTU lowered 12 bytes; both ends
   must be Linux with eBPF. State plainly it is not anti-censorship.
2. **NEW artifacts/mimic.md** — the obfuscation-artifact contract: positioning (QoS/UDP-block/
   throughput, NOT censorship); per-node model (mimic on the egress NIC, `mimic@<egress>`, config
   `/etc/mimic/<egress>.conf`, one `-f local=…:<listen_port>` filter per mimic link); MTU−12;
   provisioning (distro package, kernel eBPF + `modprobe mimic`); keyless rationale; GPL-2.0
   not-bundled note; the exact directive syntax is deferred to plan-2 (read mimic source). Link it
   from artifacts/README if an index exists.
3. **install-script.md** — document the mimic phase ordering: package/module ensure in Phase 1;
   egress detection + mimic config + `systemctl enable --now mimic@<egress>` BEFORE `wg-quick up`;
   uninstall stops/disables/removes mimic + config + modules-load and detaches.
4. **security.md** — note: mimic is protocol-shaping, not encryption or authentication; it adds NO
   secret material (WG keys remain the only secrets); obfuscation ≠ confidentiality.
5. **validation.md** — REMOVE the `tcp` reserved-warning row; add: `tcp` valid + meaningful; both
   endpoints must be Linux-deployable (platform debian/ubuntu) else error; kernel-eBPF availability
   is an install-time check (documented, not a compile error); MTU-lowering is informational.
6. **wire-contract.md** — update the transport row: `tcp` = mimic-wrapped (no new wire field).
