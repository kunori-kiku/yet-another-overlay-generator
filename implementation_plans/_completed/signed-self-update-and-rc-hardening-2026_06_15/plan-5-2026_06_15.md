# plan-5 — controller-mode UX & resilience (Unit B)

See `outline.md`. Milestone: **beta.1**. Prerequisites: plan-4 (rebases on it — shares
`frontend/src/i18n/messages/{en,zh}.ts` + `NodeRegistry.tsx`). Frontend-only (+ confirm the error envelope
shape from the backend `apierr`).

## Goal
Make controller-mode failures readable and localized, and the panel resilient — the cluster of user-visible
rough edges the RC survey flagged.

## Reads from specs
`panel-auth`, `panel-shell`, `panel-deploy-fleet` (coded-error-envelope detail: `docs/spec/api/http-api.md`).

## Read first
- `frontend/src/api/controllerClient.ts` (the `throw new Error(\`${res.status} ${body}\`)` sites — ~`:266,333,478,498,539,765`).
- `frontend/src/stores/controllerStore.ts` (English literal notices + the raw `${status} ${body}` error throws) + `topologyStore.ts`.
- `frontend/src/components/pages/*` (Fleet, Security pages — swallowed errors); dropdown `<option>` labels.
- `frontend/src/i18n/messages/{en,zh}.ts`; `frontend/src/i18n/` (tError envelope localizer).

## Implementation steps
1. **Localize the error path:** route controllerClient failures through the existing coded-envelope
   localizer (`tError`) instead of raw `${status} ${body}`; surface a localized message. No raw
   `<status> <JSON>` reaches the UI.
2. **English literals → i18n:** move hardcoded store + dropdown-`<option>` strings into `en.ts`+`zh.ts`
   (lockstep; rebased on plan-4's keys).
3. **Revoke confirmation:** the revoke-node action gets a typed/confirm dialog (no immediate fire).
4. **Surface swallowed errors** on Fleet + Security pages (show the localized error, not silent).
5. **React ErrorBoundary:** wrap the panel so a render/store exception shows a recoverable boundary, not a
   white screen.
6. `cd frontend && npm run lint && npm run build`; commit; push; PR.

## Tests produced

## Definition of done
No raw `<status> <JSON>` in any controller-mode error; zh operators see translated strings (en/zh lockstep,
tsc bijection green); revoke confirms; Fleet/Security errors surface; ErrorBoundary present; lint+build green.

## Out of scope
Backend error-envelope changes (already shipped `apierr`); the light-theme rewrite (defer/GA unless trivial).
