# plan-3 — mimic GitHub `.deb` install + signed `artifacts.json` (design PR-2)

See `outline.md`. Milestone: **beta.1**. Prerequisites: plan-2 (the `FetchSettings` struct). **D1 (manual
catalog)**, **D4 (omit when zero)**, **D8 (strict pin validation)** locked.

## Goal
Replace the silently-failing `ensure_cmd mimic mimic` with a distro-first → pinned-`.deb` → SHA-256-verify →
DKMS-deps ladder, and introduce `artifacts.json` as a **signed `bundleFiles` member** carrying the mimic
block (and a reserved/empty agent block plan-9 fills). This fixes the live mimic bug AND lays the signed-pin
rail with zero new trust primitives.

## Implementation steps
1. **`artifacts.json` emission:** write it per-node into the bundle dir; add `"artifacts.json"` to
   `bundleFiles` (export.go:141) and `allFiles` (:166). **D4:** OMIT the file entirely when `fs` has no mimic
   AND no agent catalog (keeps air-gap byte-identical — verify against plan-2 gate). Schema `{schema:1,
   mimic:{version,release_url,debs:{"<codename>-<arch>":{asset,sha256}}}, agent:{} /*reserved*/}`. Asset
   filename stored as DATA (sidesteps the mimic naming open-question).
2. **install.sh mimic block** (replace script.go:432; delete `,mimic` at :442; mirror to client template),
   gated `{{ if .HasMimic }}`: `command -v mimic || _pm_install mimic` (distro-first) → else apt/dpkg-only
   GitHub fallback: read `<codename>-<arch>` pin from `artifacts.json` (jq if present, else grep/sed — safe,
   file is integrity-verified) → empty ⇒ `fail` → `curl -fL --retry 3 --proto '=https,http' "${GH_PROXY}${release_url}/${asset}"`
   → `sha256sum` compare vs pin (FAIL-CLOSED under `set -e`) → install `linux-headers-$(uname -r)`
   (fallback `-generic`)+gcc+dkms → `apt-get install -y "$tmp.deb"` → `command -v mimic || fail`. The
   `mimic@<egress>` gate (script.go:668) is the authoritative eBPF/kernel check — unchanged.
3. **Settings:** add `MimicVersion`, `MimicReleaseBase`, `MimicDebSHA256 map[string]Artifact` to
   `ControllerSettings`/`DefaultSettings`/`WithDefaults`; validate in `HandleSettings` (D8: 64-hex SHA, semver
   version, http(s) base). Populate `FetchSettings.Mimic*` in `compile.go` from settings.
4. Commit per sub-step; push; PR.

## Definition of done

tests pass with `artifacts.json` in the digest; air-gap (no catalog) bundle byte-identical (plan-2 gate);
settings round-trip + validation.

## Stop-loss

**STOP and draft plan-3.5** (per-mode branch in export.go's node loop).
