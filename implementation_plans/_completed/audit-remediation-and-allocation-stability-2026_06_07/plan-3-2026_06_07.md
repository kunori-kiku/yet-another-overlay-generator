# Plan 3 — Compile feedback + API/allocator hardening (PR #3)

Outline: [outline.md](outline.md) · Prerequisites: plan-1 (Spec amendments to http-api.md frozen)

## Goal

Warnings reach the user on compile (UX-1 BLOCKER); the API survives hostile/degenerate input
(IPv6 CIDR panic class, oversized bodies, slow clients, handler panics); the frontend stops
swallowing server errors. Findings: UX-1, D42, D33, D34, D60 (API), D4/D20/D35/D56 (allocator),
plus nat.go/CLI garbled i18n strings.

Reads from specs: docs/spec/api/http-api, docs/spec/compiler/validation, docs/spec/compiler/ip-allocation

## Implementation steps

1. **Warnings on compile (UX-1):** add `Warnings []validator.ValidationError` to CompileResponse
   (Go + TS `CompileResponse`); HandleCompile runs `validator.ValidateSemantic` after schema pass
   and populates warnings (errors still 422). Render warnings in RightPanel compile-result
   section reusing BottomBar's warning markup; localized strings per existing i18n pattern.
2. **validate() res.ok (D42):** mirror compile()'s `if (!res.ok) throw` pattern.
3. **i18n garble:** fix nat.go:27/35/61 and cmd/compiler/main.go:40-41 message strings to match
   the locale convention from commit 0bd9dd8.
4. **Server hardening (D33/D60):** construct `&http.Server{ReadHeaderTimeout:10s, ReadTimeout:30s,
   WriteTimeout:120s, IdleTimeout:120s, MaxHeaderBytes:1<<20}`; add `recoverMiddleware` (defer
   recover → log stack → 500 JSON if header unwritten) wrapping the mux alongside cors.
5. **Body cap (D34):** `r.Body = http.MaxBytesReader(w, r.Body, 4<<20)` in readTopology (pass
   `w` in); return 413 on overflow.
6. **Allocator guards (D4/D20/D35/D56):** schema validation rejects non-IPv4 domain CIDRs and
   reserved_ranges (`ipNet.IP.To4() == nil` → error "IPv4 CIDR required" localized); defense in
   depth: `ipToUint32` returns `(uint32, error)` on len ∉ {4}; `allocateFromCIDR` bounds
   hostBits (reject CIDR larger than /8; document limit in spec) and fixes the `uint32(1)<<32`
   overflow.
7. Tests, commit, push, PR #3. Commit template: `fix(api): <step> (UX-1/D33/D34/D42/D60, D4/D20/D35/D56)`

- `internal/allocator/ip_family_test.go` — **perpetual** — IPv6/oversized CIDRs return clean
  errors (no panic), /8 boundary accepted, /7 rejected. Retirement: never.

  middleware converts an injected panic into 500 JSON. Retirement: when subject closes →
