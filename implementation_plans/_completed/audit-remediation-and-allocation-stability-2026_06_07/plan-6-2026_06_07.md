## Implementation steps

1. **Normalize + reject (D2/D72):** validation normalizes empty routing_mode → "babel"
   (write-back into the topology so it round-trips); `static`/`none` → localized validation
   ERROR "routing_mode '<v>' is reserved — not yet implemented". Same for empty
   transport → 'udp' server-side default (model normalization, part of D68's backend half).
2. **Kernel-route pairing (D40/D41) — BEHIND THE GATE:** first add the byte-identical gate test
   (render babeld.conf + install.sh for a representative router/relay/gateway/peer topology on
   UNMODIFIED code, freeze as golden). Then: for each announced prefix that is not a connected
   route, install script adds `ip route add <prefix> dev dummy0 proto static metric 4294967294`
   style anchor routes (or babel `redistribute ip <prefix>` non-local directive — choose per
   babeld manual semantics, document in Spec C) for: gateway default route, extra_prefixes,
   domain CIDR aggregate. Assert the self-/32 dummy0 path renders byte-identical to golden.
   *Stop-loss:* if CI can't prove equivalence or babeld semantics are ambiguous → ship
   gateway-default + extra_prefixes only; draft plan-6.5 for domain-CIDR aggregate.
3. **Client AllowedIPs union (D30):** DeriveClientConfigs populates DomainCIDRs from ALL
   `topo.Domains` CIDRs + every distinct resolved transit CIDR (Decision 6).
4. **Capability inference (D49):** router/gateway get `CanAcceptInbound = HasPublicIP` in
   InferCapabilitiesFromRole, mirroring DeriveRoleSemantics.
5. **Dead-edge promotion (D50):** both-ends-NAT + no endpoint + no relay path → ERROR (was
   warning); message tells the user which side needs a public endpoint or relay.
6. **Babel cleanups:** skip `IsClientPeer` interfaces (D73); Priority/Weight → per-interface
   rxcost with preset fallback (D63); wire HelloInterval/UpdateInterval presets + configurable
   local-port with current values as defaults (D78).
7. Tests, commit, push, PR #6. Commit template: `fix(routing): <step> — Spec C (D2/D30/D40/D41/D49/D50/D63/D72/D73/D78)`
