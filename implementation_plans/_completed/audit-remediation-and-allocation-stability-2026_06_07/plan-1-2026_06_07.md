## Implementation steps

1. **Spec A — Port & Endpoint Contract.** Amend `docs/spec/data-model/edge.md`: ownership table
   (backend sole port authority; `endpoint_port` is operator NAT override ONLY, never
   frontend-auto-stamped; 0/absent = auto), `compiled_port` = the port the from-side dials
   (remote port, override-aware after Plan 2). Amend `docs/spec/compiler/peer-derivation.md`:
   normative resolution rule (Endpoint emitted iff `endpoint_host != ""`; port = override if >0
   else remote allocated port; reverse-endpoint fallback from source node's public_endpoints —
   the Plan-2 target behavior), worked examples: default, NAT override, parallel edges to a hub.
2. **Spec B — Allocation Stability & Growth.** Replace the DRAFT stub
   `docs/spec/compiler/allocation-stability.md` with full normative text: I1–I10 (expand each
   from the stub's skeleton with MUST language + violation examples), the sticky-pin mechanism
   (6 `pinned_*` Edge fields, `AllocSchemaVersion`, reserve-all-pins-first then gap-fill,
   `pinKey(a,b)=sorted min|max`, hash-seeded gap-fill), key persistence rule (non-empty
   `wireguard_public_key` ⇒ fixed), migration section (Decision 7: fixed_private_key paste
   procedure OR one-final-rotation), explicit dependencies note (additive-apply + deploy
   selector are out-of-scope future subjects).
3. **Spec C — Routing Modes.** NEW `docs/spec/compiler/routing-modes.md`: enum {babel, static,
   none}; empty normalizes to `babel` at validation; `static`/`none` are REJECTED with
   "reserved — not yet implemented" (Decision 3); `Table = off` is a babel-mode artifact;
   kernel-route prerequisite rule for every announced prefix (Plan-6 target). Update
   `docs/spec/compiler/pipeline.md` + `data-model/domain.md` cross-references.
4. **Spec D — Artifact Naming & Deploy Identity.** NEW `docs/spec/artifacts/naming.md`:
   canonical installer name = `safeInstallerFileName(node.Name)` everywhere (ZIP entry, deploy
   lookup, remote path uses node ID); uniqueness invariant (raw, sanitized, and wgInterfaceName
   collisions are validation ERRORS); de-collision = short node-ID hash suffix; wgInterfaceName
   algorithm including the >15-char `clean[:8]+sha256[:4]` branch; frontend MUST consume
   compiled interface names, never reconstruct.
5. **Spec E — Wire Contract.** NEW `docs/spec/api/wire-contract.md`: field-parity table
   (Go model vs TS vs UI vs round-trip), `route_policies` = **RESERVED** (validator rejects
   non-empty; Decision 2), `transit_cidr` = exposed (Plan 9), key-blanking round-trip semantics
   (non-fixed keys blanked in response — by design pre-Plan-7, pin-reuse after), rule: every
   compile-response topology field the FE persists.
6. **Amendments.** `docs/spec/api/http-api.md`: compile MUST run semantic validation and return
   `warnings`; body-size limit + server timeouts (Plan-3 targets). `docs/spec/compiler/validation.md`:
   coverage table — every model field × validated-where (the T4 gaps marked as Plan-5/9 targets).
   `docs/spec/data-model/node.md` + `docs/spec/security/security.md`: key persistence semantics.
   `docs/spec/compiler/ip-allocation.md` + `peer-derivation.md`: determinism caveats (order-dependence
   facts until Plan 7 lands).
7. **CI workflow.** NEW `.github/workflows/ci.yml`:
   ```yaml
   name: ci
   on: { pull_request: {}, push: { branches: [main] } }
   jobs:
     go:
       runs-on: ubuntu-latest
       steps: [checkout, setup-go (from go.mod), go vet ./..., go test ./...]
     frontend:
       runs-on: ubuntu-latest
       defaults: { run: { working-directory: frontend } }
       steps: [checkout, setup-node 20, npm ci --legacy-peer-deps, npm run lint, npm run build]
   ```
   Mirror release.yml's setup-go/setup-node versions.
8. **Commit + PR.**
   ```
   git checkout -b plan/audit-remediation-2026-06
   git add docs/spec docs/audit implementation_plans STATUS.md PRINCIPLES.md docs/DEVELOPMENT_SPEC.md .github/workflows/ci.yml
   git commit -m "docs(spec): restructure spec into docs/spec components; freeze contracts; add audit dossier, plan folder, PR CI"
   git push -u origin plan/audit-remediation-2026-06
   gh pr create --title "Contract freeze: docs/spec restructure, normative specs A-E, audit dossier, PR CI" --body "<summary + dossier link + decisions log pointer>"
   ```
9. If ci.yml is RED on this docs-only PR (pre-existing test failures on main) → STOP, update
