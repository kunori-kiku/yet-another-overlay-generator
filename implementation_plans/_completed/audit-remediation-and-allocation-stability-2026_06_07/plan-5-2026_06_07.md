## Goal

No user-controlled text reaches a shell context unescaped; the self-extracting installer verifies
its payload before root execution; effective ports validated. Findings: D7, D43, D44, D15, D16
(injection), D24, D25 (integrity), D11, D47 (ports).

Reads from specs: docs/spec/artifacts/naming, docs/spec/artifacts/deploy-scripts,
docs/spec/artifacts/install-script, docs/spec/compiler/validation, docs/spec/security/security

## Implementation steps

1. **Shell quoting (D7/D43):** add `bashQuote(s string)` (single-quote escape `'\''`) and
   `ps1Quote` helpers in `internal/renderer/escape.go`; apply to SSHTarget at every bash site
   (282,286,304,311,324,325) and ps1 site (507,514,530,537,540,545). Mirror style of existing
   SSHKeyPath %q.
2. **SSH field validation (D44):** `validateSSHFields` in semantic.go — charset
   `^[A-Za-z0-9._:@-]+$` for ssh_host/ssh_user/ssh_alias (error, localized); matching `pattern`
   attrs on the RightPanel SSH inputs.
3. **Node-name safety (D15/D16):** schema validation restricts node.Name to
   `^[A-Za-z0-9 ._-]+$` (no quotes/backticks/$); AND escape NodeName at template injection
   sites (script.go echo contexts, deploy.go heredoc) via the same helpers — defense in depth.
4. **Integrity chain (D24/D25):** export.go adds install.sh to checksumLines; handler.go
   `makeSelfExtractingInstaller` computes SHA-256 of the tar.gz in Go, embeds
   `EXPECTED_SHA256=<hex>` in the wrapper, wrapper verifies after decode and aborts before
   extract/run on mismatch.
5. **Port bounds (D11/D47):** DerivePeers errors when any effective port >65535;
   `validateListenPortConflicts` computes per-node effective range (base..base+ifaceCount-1),
   errors on cross-node range overlap for same hostname and on range exceeding 65535.
6. Tests, commit, push, PR #5. Commit template: `fix(security): <step> (D7/D11/D15/D16/D24/D25/D43/D44/D47)`

## Tests produced by this plan

- `internal/renderer/escape_test.go` — **perpetual** — hostile fixtures (`x; rm -rf $HOME #`,
  `Bob's Router`, `$(touch /tmp/pwned)`) render into scripts containing NO unquoted
  metacharacters (assert rendered output quotes them inert); validator rejects them where
  charset rules apply. Retirement: never (pins the root-scripts principle).
- `internal/artifacts/export_integrity_test.go` — **perpetual** — install.sh present in
  checksums.sha256; wrapper contains the correct payload hash and the verification line.
  Retirement: never.
- `internal/compiler/peers_portbounds_test.go` — **subject-scoped** — base 65530 + 8 peers →
  compile error; co-hosted overlapping ranges → validation error. Retirement: fold the
  overflow case into peers tests at closure.
