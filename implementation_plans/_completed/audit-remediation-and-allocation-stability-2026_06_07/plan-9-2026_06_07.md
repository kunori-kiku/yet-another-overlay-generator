1. **route_policies reserved (Decision 2):** schema validation errors on non-empty
   `route_policies` ("reserved — not implemented", localized); TS type comment marks it
   reserved; importProject warns + strips with a visible notice (no silent drop, D45/D55).
   Keep the Go struct (wire-compat) — no renderer work (D37 closed by the reject).
2. **transit_cidr exposure (D46):** TS Domain gains `transit_cidr?: string`; DomainForm +
   domain editor input with CIDR validation; survives round-trip (getTopology already
   serializes typed fields once the type carries it — verify).
3. **Validation coverage (D64–D67):** MTU (0 or 576–65535), SSHPort (0 or 1–65535), RouterID
   (MAC-48 regex or IPv4), ExtraPrefixes (each net.ParseCIDR) — schema errors, localized;
   matching min/max/pattern on FE inputs.
4. **Edge fields (D68/D71):** RightPanel edge editors for transport/priority/weight/notes;
   backend defaults empty transport→'udp' (normalization from Plan 6 — verify, extend);
   semantic warning for duplicate enabled edges on one pair.
5. **Role/caps coherence (D54/D69/D26):** role onChange re-derives can_forward/can_relay/
   can_accept_inbound for ALL roles (mirror NodeForm + roles.go inference); NodeForm offers
   'client'; AuditView exposed-nodes reads `compileResult.topology.nodes` (backend-inferred)
   with a "recompile to refresh" hint when store differs.
6. **State bugs:** `crypto.randomUUID()` for edge/node/domain/endpoint IDs (D17); position-sync
   useMemo→useEffect with honest deps (D18); AuditView selector value encoding via
   `encodeURIComponent` + `|` delimiter (D58); preview `{'\n'}` fix (D74); loadTopology clears
   history+selection (D75); diff keeps 3 context lines + skipped-count label (D77).
