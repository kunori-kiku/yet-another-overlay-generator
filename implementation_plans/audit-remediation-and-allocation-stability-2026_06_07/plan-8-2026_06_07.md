## Implementation steps

1. **Extract:** new package `internal/render` (or `internal/compiler` extension — pick the one
   that avoids import cycles; renderer imports model only, handler imports both → `internal/render`
   is clean): `render.GenerateKeys(topo) (KeyMap, error)` and
   `render.All(result *compiler.CompileResult, keys KeyMap) error` carrying the full handler
   logic (per-peer + client WG configs, babel, sysctl, install incl. client branch, deploy
   scripts). Handler and CLI both call it. Descriptive names — no "shared"/"common" vagueness.
2. **CLI:** delete `generateFakeKeys` entirely (D6); `cmd/compiler/main.go` calls
   render.GenerateKeys (honors FixedPrivateKey via wgtypes.ParseKey) + render.All; client nodes
   get wg0.conf + client install script (D27/D28/D29); deploy scripts written (D59). Keep a
   `-skip-keys` flag ONLY if Spec says so — default is real keys, no silent placeholders.
3. **Install robustness:** iptables SNAT cleanup loop-deletes every POSTROUTING rule matching
   `-o wg-+ -s <transitCIDR>` regardless of --to-source (parse iptables-save) (D52);
   `wg-quick up` per-interface failure collected (`|| FAILED_IFACES="$FAILED_IFACES $iface"`),
   babeld still configured/started, summary reported at end, nonzero exit if any failed (D53).
4. **README architecture string (D76):** use the manifest's architecture variable for client
   bundles.
5. Tests, commit, push, PR #8. Commit template: `refactor(render): unified entrypoint + install robustness (D6/D27-29/D52/D53/D59/D76)`
