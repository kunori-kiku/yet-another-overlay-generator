## Implementation steps

1. **Default domain (UX-3):** initial store state seeds
   `{id:'domain-default', name:'overlay', cidr:'10.20.0.0/24', allocation_mode:'auto',
   routing_mode:'babel'}` (NOT 10.10.0.0/24 — transit collision); remove the disabled guard on
   Add Node (NodeForm:68); DomainForm remains for editing/adding more.
2. **Visible handles + connection validity (UX-4):** pre-compile fallback handles in CustomNode
   become role-colored (reuse roleColors map), larger, with hover-grow + title "drag to
   connect"; `<ReactFlow isValidConnection>` rejects self-loops and duplicate enabled pairs.
3. **Public address surfacing (UX-5):** NodeForm gains a top-level "Public address (IP:port) —
   how peers reach this server" input; non-empty value derives has_public_ip +
   can_accept_inbound and creates public_endpoints[0] (the checkbox becomes derived/advanced);
   rename "endpoint mappings" labels to plain language (NodeForm:147, RightPanel:369); client
   role keeps the block hidden (correct behavior, not a bug).
4. **extra_prefixes editor (UX-6):** list editor in RightPanel mirroring the endpoint-list
   pattern (:62-92), CIDR-validated entries, visible for gateway always / router+relay with a
   hint, warning for roles where it's a no-op (roles.go:51,64,77 gating).
5. **Docs sync (T14):** README:69 + wiki:110 remove "Auto:<port> button" (describe compiled_port
   display + explicit NAT override); wiki role tables add extra_prefixes for router/relay +
   client row (wiki:195); document babel hello/update timers + naming algorithm hash branch;
   verify docs/spec matches shipped behavior end-to-end (final reconciliation pass).
6. **Closure ritual:** outline closure checklist; map every dossier finding ID →
   fixed-in-PR / deferred-by-decision / superseded (append table to dossier); retire
   subject-scoped tests per their triggers; STATUS.md refresh; `git mv` subject folder to
   `implementation_plans/_completed/`; final commit.
7. Commit template: `feat(ux): bridging affordances + docs sync (UX-3/4/5/6, T14) — subject closure`
