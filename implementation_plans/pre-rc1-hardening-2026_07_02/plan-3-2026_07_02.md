## Implementation steps

### Step 1 — hygiene: remove the stray binary + gitignore
```
git rm --cached agent
```
Add to `.gitignore` (root) a build-artifacts block covering the top-level binaries produced by the
documented build commands:
```
# top-level build artifacts (produced by `go build -o <name> ./cmd/...`)
/agent
/server
/compiler
/e2eserver
/e2eagent
/yaog-server
/yaog-compiler
```
(Leading `/` anchors to repo root so it never masks a legitimately-named source dir elsewhere.)
Confirm `git status` shows `agent` deleted-from-index + `.gitignore` modified, and that the other
untracked binaries are now ignored.

### Step 2 — roll the CHANGELOG
Move `## [Unreleased]` → `## [2.0.0-beta.17] - 2026-07-02` (adjust date at cut time); add a fresh empty
`## [Unreleased]`; add the compare link at the file bottom. The section covers `beta.16..HEAD` — the two
criticals + the hygiene cleanup. Group under `### Fixed` (NAT override, self-update keystone bypass) and
`### Changed`/`### Removed` (stray binary). Confirm with `git log v2.0.0-beta.16..HEAD --oneline`.
