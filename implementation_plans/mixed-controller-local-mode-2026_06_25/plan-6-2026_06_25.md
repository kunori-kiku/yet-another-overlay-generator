## Goal

Surface the manual-node feature in the panel: mark a node manual + provide its pubkey/endpoint, let a
manual node's PUBLIC key survive controller import, show manual nodes as "manual/unmonitored"
(excluded from convergence), and download a manual node's signed bundle for hand-delivery.

## Implementation steps

### Step 1 ‚Äî Types + editor

- `topology.ts` ‚Äî `deployment_mode?: 'managed' | 'manual'` on `Node`.
- `NodeEditor.tsx` / `NodeForm.tsx` ‚Äî in CONTROLLER mode, a "Deployment" control (Managed | Manual).
  When Manual: show inputs for the node's `WireGuardPublicKey` (paste, validated) + `PublicEndpoints`/
  edge endpoint, with helper text explaining the kit (plan-4) can generate these on-box. Managed is
  the default; existing nodes are unaffected.

### Step 2 ‚Äî Custody relax (public allowed for manual)

`custody.ts` ‚Äî when importing/serializing a controller topology, allow a manual node's
`WireGuardPublicKey` to survive (it must reach the server so plan-1 admission can read it) while
`dropAllKeys`/`stripPrivateKeys` continue to strip EVERY private key (`fixed_private_key`,
`WireGuardPrivateKey`) for all nodes. Add a unit test pinning: manual node keeps pubkey, loses any
private; managed node loses all key material (unchanged).

### Step 3 ‚Äî Manual-node panel UX

- A "manual / unmonitored" chip on the node (registry table + detail page), visually distinct from
  the enrolled/approved status chips (use the tokenized status families from the theme subject ‚Äî a
  neutral `--control`/`--info` treatment, not a health color).
- **Convergence exclusion (D3):** the "fleet converged" rollup excludes manual nodes; surface a small
  "N manual node(s) ‚Äî not monitored" note instead of counting them as drift.
- **Per-node bundle download (plan-3):** a "Download bundle" action on a manual node ‚Üí calls
  `downloadManualNodeBundle(nodeId)`.
- If the manual node opted into the reporter (plan-5), render its conditions like any other node's
  (the existing nodeConditions chips ‚Äî now tokenized from the theme subject).

### Step 4 ‚Äî Register-manual-node action

A panel flow to register a manual node's pasted descriptor (`{NodeID, pubkey, endpoint}`) via
`registerManualNode` ‚Üí `POST /api/manual-node` (plan-2).

  `internal/localcompile` golden contract unaffected.

## Hazard

The custody relax is the one place a private key could leak into localStorage/the server if done
wrong. Adversarially verify: ONLY the public key of a manual node survives; every private key field
(`fixed_private_key`, `WireGuardPrivateKey`) is still stripped for every node; the leakOracle e2e
backstop stays green.
